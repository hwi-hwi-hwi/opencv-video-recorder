# 🎥 OpenCV Video Recorder

💻 A real-time webcam video recorder implemented with OpenCV in Python.

## ✨ 기능 설명 (Features)
- ⏯️ **Preview / Record 모드**: `Space` 키를 눌러 녹화 모드를 토글할 수 있습니다.
- 🔴 **녹화 표시 (Recording Indicator)**: 녹화 중일 때는 화면 좌측 상단에 텍스트 테두리 효과가 적용된 뚜렷한 `REC` 글자와 빨간색 원이 표시됩니다.
- 💾 **동영상 저장**: 녹화된 영상은 `output.avi` 파일로 실시간 저장됩니다.
- 🪞 **추가 기능 (Flip Filter)**: `F` 키를 누르면 영상이 좌우로 반전(Horizontal Flip)되는 필터 기능이 작동합니다.
- 🚪 **종료**: `ESC` 키를 누르면 프로그램과 카메라가 안전하게 종료됩니다.
